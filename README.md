# AI-Based Text Summarization System

An AI-powered Text Summarization Web Application developed using Flask, NLP, and Hugging Face Transformers. The system generates concise summaries from long text using Extractive and Abstractive summarization techniques.

---

## Features

- Extractive Text Summarization
- Abstractive Text Summarization
- AI-powered summarization using Transformers
- Interactive Web Interface
- Flask Backend Integration
- Responsive UI Design

---

## Technologies Used

### Frontend
- HTML
- CSS
- Bootstrap
- JavaScript
- jQuery

### Backend
- Python
- Flask
- Flask-CORS

### AI / NLP
- Hugging Face Transformers
- BART / DistilBART Model
- NLTK
- PyTorch

---

## Project Workflow

1. User enters long text.
2. Frontend sends request to Flask backend.
3. Backend processes text using NLP.
4. Extractive summary is generated.
5. Transformer model generates abstractive summary.
6. Final summaries are displayed to the user.

## Installation

### Clone Repository

```bash
git clone https://github.com/YOUR_USERNAME/text-summarization-project.git
```

### Open Project Folder

```bash
cd text-summarization-project
```

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
python app.py
```
